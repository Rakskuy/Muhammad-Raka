HyperText Markup Language (HTML)
HyperText Markup Language adalah bahasa markup standar yang digunakan untuk membuat atau menyusun struktur konten di suatu website. HTML memberikan struktur dasar untuk halaman web dengan menggunakan elemen-elemen yang ditandai dengan simbol tag < >. Setiap tag elemen HTML dapat memiliki atribut yang memberikan informasi tambahan tentang elemen tersebut.
